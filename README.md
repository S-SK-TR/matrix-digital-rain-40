# Matrix Dijital Yağmur

Matrix filminden esinlenerek oluşturulan dijital yağmur ekranı animasyonu.

## Özellikler

- Canvas API kullanarak performanslı animasyon
- Kullanıcı kontrol paneli ile etkileşimli deneyim
- Responsive tasarım ve mobil uyumluluk
- Hız, yoğunluk, renk gibi parametreleri ayarlama

## Kurulum

1. Depoyu klonlayın
2. `npm install` komutunu çalıştırın
3. `npm run dev` ile geliştirme sunucusunu başlatın

## Kullanım

- Animasyon parametrelerini sağ alt köşedeki kontrol panelinden ayarlayabilirsiniz

## Testler

`npm test` komutu ile testleri çalıştırabilirsiniz.
